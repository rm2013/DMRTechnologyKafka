To run the Calendar Service: 
~~~~
cd servers
cd calendarservice
mvn spring-boot:run
~~~~

To run the Calendar App: 
~~~~
cd servers
cd calendarapp
mvn spring-boot:run
~~~~

To run the client:
~~~~
cd client
npm install
yarn serve
~~~~
